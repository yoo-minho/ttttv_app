#예능 프로그램 채팅 앱 <티티티티비> 소스입니다.

#구현한 기능

- 로그인 & 회원가입
- 채팅방 (TCP/IP 활용, 페이징, 알림, 검색, 방나누기, 크롤링)
- 랜덤영상통화 (WebRTC 활용)
- 연예인닮은꼴찾기 (NaverCFR api + OPEN CV)

#활용한 라이브러리

- Retrofit(http통신)
- Firebase(Google Analytics)
- Jsoup(크롤링)
- Glide(이미지)
- StyleableToast(스타일토스트)
- WebRtc(영상통화)
- OpenCV(얼굴검출)
- NaverCFR(이미지인식)
